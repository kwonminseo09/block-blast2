프로젝트명: Block Blast Master (Web Edition)
1. 목적
사용자 경험: 중독성 있는 퍼즐 메커니즘과 부드러운 시각 효과를 통해 사용자에게 즐거움 제공.

기술적 목표:

Canvas API 또는 DOM 조작을 통한 고성능 그래픽 렌더링 구현.

복잡한 배치 알고리즘 및 게임 루프(Game Loop) 로직 설계 능력 강화.

2. 필수 기능
지능형 블록 생성 시스템:

단순 무작위가 아닌, 현재 보드 상황에 따라 난이도가 조절되는 블록 생성 로직.

3개의 블록을 한 세트로 하며, 모두 소비 시 새로운 세트 생성.

고급 드래그 앤 드롭:

마우스 및 터치 이벤트(Touch Event) 동시 지원.

블록을 들었을 때 보드 위에 예상 배치 위치를 미리 보여주는 고스트(Ghost) 가이드.

콤보 및 스코어링 시스템:

한 번에 여러 줄(가로+세로)을 동시에 지울 시 가산점 부여.

연속으로 줄을 지울 경우 콤보 스택 적용 및 화려한 UI 연출.

게임 데이터 관리:

LocalStorage를 이용한 최고 점수(High Score) 및 현재 게임 상태 자동 저장.

브라우저를 껐다 켜도 마지막 상태에서 이어서 하기 지원.

게임 오버 밸런싱:

블록을 놓을 곳이 없는지 매 턴마다 자동으로 체크하여 즉시 게임 오버 처리.

3. 디자인 요구 (Visual & UX)
다크 모드 & 네온 테마: 눈이 편안하면서도 몰입감 있는 다크 테마와 세련된 네온 컬러 블록.

애니메이션 효과 (CSS Transition & Canvas):

줄이 완성될 때 블록이 사방으로 흩어지거나 반짝이며 사라지는 파티클 효과.

블록을 배치할 때 발생하는 미세한 화면 흔들림(Screen Shake) 효과.

사운드 디자인:

블록 선택, 배치, 라인 제거(콤보마다 음정 상승), 게임 오버 시 효과음 삽입.

배경음악(BGM) On/Off 토글 기능.

반응형 웹: 모바일, 태블릿, PC 어디서든 최적의 비율로 보이는 유연한 레이아웃.

4. 기술 스택
Core: HTML5 (Canvas 활용 권장), CSS3 (Flexbox/Grid, Animations), JavaScript (ES6+).

Tools:

Vite 또는 Live Server (빠른 개발 환경).

Web Audio API (정교한 사운드 제어).

Deployment: GitHub Pages 또는 Vercel (무료 웹 호스팅).
